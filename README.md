# campuspe-cybsec-assignment
Description

This repository contains simple bash scripts for:

System information (q1_system_info.sh)

File and directory management (q2_file_manager.sh)

Log file analysis (q3_log_analyzer.sh)

Automated backups (q4_backup.sh)

User account reporting (q5_user_report.sh)

Files

sample.log – sample log file

incident_report.txt – output of log analysis

user_report.txt – output of user report

backup/ – folder containing backups

How to run

Make scripts executable:

chmod +x *.sh

Run any script:

./q1_system_info.sh

Replace with the script you want to run.
